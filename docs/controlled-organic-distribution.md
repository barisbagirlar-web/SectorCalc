# Controlled Organic Distribution

Organic-first growth rhythm before paid micro-tests.

**Prerequisite:** [Controlled Scale Sprint](./controlled-scale-sprint.md) — organic always allowed if no mobile/checkout blocker.

---

## Weekly rhythm

| Activity | Target |
|---|---:|
| LinkedIn posts | 3 |
| Community posts (Reddit/niche) | 2 |
| Direct outreach messages | 20 |
| GSC URL inspections | 5 |
| Internal link improvements | 5 |
| Authority content refresh | 1 |

Log cluster + URL in [Controlled Scale Sprint Report](./controlled-scale-sprint-report.md).

---

## Content priority

1. **Hidden loss** — scrap, overrun, drift, exposure (cluster `primaryPain`)
2. **Calculator use** — show the free tool outcome, not feature lists
3. **Decision report** — premium preview value (locked state screenshot OK)
4. **Beta feedback request** — missing variables, formula fit
5. **Comparison against spreadsheets** — Excel/manual estimate gap (no fake ERP claims)

---

## Channel guidelines

| Channel | Focus |
|---|---|
| LinkedIn | Operator audience — manufacturing, construction, logistics |
| Reddit | Niche subreddits — problem-first, not promotional spam |
| Direct outreach | Beta partner template — one cluster per week |
| GSC | Tier-1 URLs from [week-1-priority-url-matrix](./week-1-priority-url-matrix.md) |
| Internal links | Hub → free tool → premium analyzer within same cluster |

---

## Do not publish

- Fake case study
- Fake user count
- Guaranteed savings or ROI
- Fake certification
- Exaggerated revenue claim
- “#1 calculator” without evidence

Align with [Launch Campaign Content Pack](./launch-campaign-content-pack.md) tone.

---

## Success signals (weekly)

| Signal | Action if weak |
|---|---|
| Landing views up | Continue same cluster |
| Opens but no calculate | Switch to [CTA scale mode](./controlled-scale-modes.md) |
| Calculate but no premium click | CTA + locked state iteration |
| Beta submits | Route to proof page backlog item |

---

## Relationship to paid

Organic runs **always** in parallel with at most one paid micro-test.  
Paid does not replace organic — it validates intent on a single cluster only.

See [Paid Micro-Test Runbook](./paid-micro-test-runbook.md) when readiness gate passes.
