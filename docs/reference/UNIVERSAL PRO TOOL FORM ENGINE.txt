	9	PASS/FAIL generic rule:
Eğer result.status varsa:
	•	PASS → badge-pass / ok
	•	FAIL → badge-fail / crit
Eğer result.UC varsa:
	•	UC <= 0.90 → pass
	•	0.90 < UC <= 1.00 → warn
	•	UC > 1.00 → fail
UC > 1.00 ise calculation sonucu FAIL kabul edilecek. UC > 0.90 && UC <= 1.00 ise engineering review warning gösterilecek.

	16	RC Beam formulas:
EC2 formulas:
	•	ρ_l = A_s / (b_w × d)
	•	f_cd = α_cc × f_ck / γ_c
	•	f_yd = f_yk / γ_s
	•	λ = 0.8 for f_ck ≤ 50 MPa
	•	η = 1.0 for f_ck ≤ 50 MPa
	•	x = A_s × f_yd / (η × f_cd × b_w × λ)
	•	z = min(d − 0.5 × λ × x, 0.95d)
	•	M_Rd = A_s × f_yd × z
	•	k = min(1 + sqrt(200 / d), 2.0)
	•	ρ_l,limited = min(A_s / (b_w × d), 0.02)
	•	V_Rdc = max(C_Rdc × k × (100 × ρ_l × f_ck)^(1/3) + 0.15σ_cp, v_min + 0.15σ_cp) × b_w × d
	•	V_Rds = A_sw/s × z × f_ywd × cotθ
	•	V_Rdmax = b_w × z × v1 × f_cd / (cotθ + tanθ)
	•	V_Rd = min(V_Rds, V_Rdmax)
	•	UC_flexure = M_Ed / M_Rd
	•	UC_shear = V_Ed / V_Rd
	•	UC = max(UC_flexure, UC_shear)
ACI formulas:
	•	ρ_l = A_s / (b_w × d)
	•	β1 = ACI concrete stress block factor
	•	a = A_s × f_y / (0.85 × f'_c × b_w)
	•	c = a / β1
	•	ε_t = 0.003 × (d − c) / c
	•	φ_f = ACI flexural strength reduction factor
	•	M_n = A_s × f_y × (d − a/2)
	•	φM_n = φ_f × M_n
	•	V_c = 0.17 × λ × sqrt(f'_c) × b_w × d
	•	V_s = A_v × f_y × d / s
	•	φV_n = φ_v × (V_c + V_s)
	•	UC_flexure = M_Ed / φM_n
	•	UC_shear = V_Ed / φV_n
	•	UC = max(UC_flexure, UC_shear)
	17	RC Beam validation rules:
BLOCK:
	•	b_w <= 0
	•	h <= 0
	•	d <= 0
	•	d >= h
	•	A_s <= 0
	•	M_Ed < 0
	•	V_Ed < 0
	•	A_sv > 0 and s_v missing
	•	EC2 selected and f_ck missing
	•	ACI selected and f_c_prime missing
	•	UC > 1.00
WARN:
	•	UC > 0.90 && UC <= 1.00
	•	stirrup spacing appears too large
	•	EC2 V_Rds > V_Rdmax
	•	ACI V_s > V_s_max
	•	ACI epsilon_t < 0.005
	•	EC2 x/d > 0.45
