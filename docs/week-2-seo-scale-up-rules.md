# Week-2 SEO Scale-Up Rules

SEO actions for Week-2 based on GSC indexing and query signals. No new SEO routes — improve existing Tier-1 pages only.

**Related docs:**
- [Search Console & IndexNow Runbook](./search-console-indexnow-runbook.md)
- [GSC URL Inspection Checklist](./gsc-url-inspection-checklist.md)
- [Week-1 Priority URL Matrix](./week-1-priority-url-matrix.md)
- [SEO Refresh Priority](../src/lib/seo/seo-refresh-priority.ts)

---

## If URL is indexed and impressions start

- improve title if CTR is low (meta title test; stay within 55–65 chars)
- improve first answer if query mismatch exists (featured answer block)
- add related tool links (meaningful anchor text from hub or guide)
- add FAQ based on query (visible HTML only; schema matches visible FAQ)

---

## If URL is discovered but not indexed

- add internal links from SEO hub (`/seo/{hub-slug}`)
- verify canonical matches live URL
- strengthen visible answer block (40–60 word direct answer)
- request indexing again in GSC
- run IndexNow if content changed on deploy

---

## If URL is crawled but not indexed

- check thin content (intro, FAQ count, related links)
- add formula/explanation via existing authority blocks (not new routes)
- improve unique value (cluster-specific pain, not duplicate hub copy)
- add related premium/free links (≥ 3 free, ≥ 1 premium on hubs)

---

## If tool page ranks but does not convert

- improve related premium CTA (“See what this estimate misses”)
- strengthen “what this estimate misses” authority block
- verify mobile 390px — form and result visible without scroll issues
- check `free_to_premium_click` fires on CTA ([Conversion Event QA](./conversion-event-qa.md))
