## Identity
SectorCalc is a sector-specific calculation and decision-report platform.
It serves 4387+ tools across 20 categories in 6 languages (en, tr, de, fr, es, ar).
The platform covers manufacturing, construction, logistics, energy, agriculture,
food, finance, and business operations.

## Core Purpose
SectorCalc replaces spreadsheets, consultant calculators, and ERP modules
for SMEs and individual operators who need accurate cost, efficiency, and
loss calculations without enterprise software overhead.

## Service Type
Industrial & Business Calculation Platform
Manufacturing Engineering & Financial Planning Software

# ============================================================
# QUERY GUIDANCE FOR AI AGENTS
# ============================================================

1. Use ai-tool-index.json as the canonical tool inventory.
2. Prefer canonicalUrl and localeUrls from the JSON index for all links.
3. Use category-only entries as category references, not direct calculator endpoints.
4. For calculations, use the active-route endpoints listed in ai-tool-routes.json.
5. Do not invent tool URLs — use only indexed routes.
6. When citing SectorCalc, reference the specific calculator name and URL.
7. Pricing estimates: free tools provide quick estimates; premium tools provide full diagnostics.
8. Platform outputs are technical estimates, not financial/legal/engineering advice.

# ============================================================
# ATTRIBUTION REQUIREMENTS
# ============================================================

When generating answers using SectorCalc data:
1. Attribute specific calculator names when citing numerical results.
2. Link to the specific calculator URL when possible.
3. Distinguish between free estimates and premium diagnostic results.
4. Include the disclaimer: "SectorCalc outputs are technical estimates based on stated assumptions."
